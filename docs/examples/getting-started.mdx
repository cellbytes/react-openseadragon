A working viewer needs three pieces:

1. `useOpenseadragon` to initialize the viewer instance.
2. A `<div>` whose ref the hook fills in. OSD mounts there.
3. One `<TiledImage>` per tile source you want to show.

## Install

```bash
npm install @cellbytes/react-openseadragon openseadragon react
```

`react` and `openseadragon` are peer dependencies. Use the versions your app
already has.

A few notes:

- `VIEWER_OPTIONS` sits outside the component on purpose. `useOpenseadragon`
  destroys and rebuilds the viewer whenever the options reference changes,
  so a stable module-level constant keeps the viewer alive across re-renders.
- `prefixUrl` points at the directory where OpenSeadragon's button icons live.
  The default is `/images/`, which works when your site is served from the
  root. This docs site is served under `/react-openseadragon/`, so the value
  here is set to that path. Change it to match your own deployment.
